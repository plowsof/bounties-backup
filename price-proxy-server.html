Price Proxy Server

Posted by
rimuru
3 months ago
Description

Not sure if this exists. Created a sample node.js server that provides XMR -> BTC price data over i2p. Sample client code as well. Not sure how much benefit it can provide for increasing privacy as opposed to fetching price over clearnet. It seems like the more the merrier. The example uses API key from cryptocompare (they have a free tier). Perhaps in the future it could aggregate price data from atomic swap sellers, to have a decentralized price feed.

Incentive:

Also, implements TPAT (transaction proof authentication token) proof of concept which is part of another bounty https://bounties.monero.social/posts/11/authentication-token-proxy

Why use tx proof as authentication token?

  • allows monetization of APIs via tx proof
  • scaling
    • payment available as soon as it leaves the tx pool
    • re-use / sharing of TPAT (like a ticket) until it expires. Expiration is based on confirmations threshold. Basically higher payment for longer access.
    • create caching payment strategies and tier-based payment for API's
    • security - no access without valid proof

See WIP:

https://github.com/hyahatiph-labs/infosec/tree/main/xmr-price-proxy

Minimum Requirements:

  • user can host a price proxy server over i2p with xmr -> btc price
  • user can receive payment and authenticate clients with a tx proof
  • code is open source on github

Payout Criteria

  • verified by other users / developers
