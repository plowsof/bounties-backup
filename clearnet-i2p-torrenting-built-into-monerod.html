Clearnet + I2P torrenting built into monerod

about 4 hours ago carrington
Description

Presently, distribution of the core Monero software mostly relies on getmonero.org or github. Now that we have reproducible builds, we should work towards distributing this critical software in a decentralized way. By giving monerod the capability to seed the source code + compiled software, Monero nodes will become self-replicating and the Monero network will become more resilient.

The bounty requirements:

  1. Submit a PR enabling torrenting (over clearnet and i2p) of the core Monero software (monerod + monero-cli + monero-gui) from within monerod.

Bonus points:

Similar capabilities within the Monero GUI or Feather.

Automatic updates via the torrenting mechanism.

Options for torrenting either/both the source code & compiled software for all supported platforms.

I am not an expert in Monero or I2P, so please comment if I have missed any critical barriers or requirements which should be specified.
