Description

If the donation address is parsed as a link users will be able to more easily copy the address and can open their monero wallet directly from the site.

The code of this site will be improved, which is free software, meaning anyone can utilize or modify it.

This bounty is intended to decentralize the development of the Monero Bounties software and relevant dependencies.

https://github.com/russross/blackfriday/issues/692

Github.com/getfider is the upstream project that will need to consume the new version of blackfriday
