Android GUI Monero Node

Posted by
cryptogrampy@protonmail.com
5 months ago
Description

There are Termux scripts for running Monerod on Android (https://github.com/CryptoGrampy/android-termux-monero-node), but I would like to see a GUI that makes this
process even easier. There are millions of phones out there that are capable of running nodes and
a GUI Monero node helps bring those phones into the network.

Minimum Requirements:

  • Must be an Android app/downloadable .apk
  • Allows custom configs or startup flags
  • Have a sane default config - default to pruned node(?)
  • Detect drive capacity/allow users to choose internal/external SD, prevent them from running monerod if they don't have adequate storage
  • Have a Button to start/stop the node
  • Display node stats (current number of peers/connections, sync progress, storage remaining on phone)
  • Informs users that their node is out of date
  • Can start the app/node on boot
  • Comes Bundled with latest ARMv7/v8 binaries
  • Detects CPU architecture and runs the correct monerod binary
  • Must have a build script for anyone to build the .apk themselves
  • Must have downloadable .apk to install
  • Code must be open source/anyone can use it/fork it/do whatever they want with it

Reach-

  • Available for download on FDroid/Play Store
  • Can update Monerod from app
